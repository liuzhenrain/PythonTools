#导出Excel数据到Sqlite3中
- 确定excelfile文件夹和ExcelToSqlite.py在同一级目录下
- 参照excelfile文件夹下excel文件的格式。
    请参照如下格式
    
    | 说明     | 说明        | 说明     | 说明 |
    | ------- | ------------- | --------- |-----|
    | 类型    | 类型        | 类型     | 类型 |
    | 列名    | 列名        | 列名     | 列名 |
    | 数据    | 数据        | 数据     | 数据      |
